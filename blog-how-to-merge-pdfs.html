Why merge PDFs in the first place?

Sending multiple files when one would do creates friction. Recipients have to open each file separately, things get lost in email threads, and file names like report_final_v3_UPDATED.pdf start multiplying. A single merged PDF is cleaner, easier to share, and simpler to archive.
